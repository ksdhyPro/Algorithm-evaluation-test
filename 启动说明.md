# web 说明

1. 前端代码若非调试，就不用管，直接启动 python 就行 python 已经托管 web 目录 访问 http://localhost:5000
2. 如果修改前端代码，修改完直接 npm run build 就行，会自动 build 到 python 托管的 web 目录，不用任何处理
3. 首次运行注意下是否存在 web 目录以及目录是否有内容 如果没有就得执行 npm run build

# 服务端说明

1. 运行可以采用 docker 部署（docker run -d -p 5000:5000 -v /var/run/docker.sock:/var/run/docker.sock my-app），也可以直接启动 python app.py
2. 记得调整 env 文件参数配置，根据需求自行配置
